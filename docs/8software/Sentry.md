# Sentry

错误监控，性能监控

# 安装

[官方文档](https://github.com/getsentry/onpremise)

需要的硬件要求:

```
Docker 19.03.6+
Compose 1.28.0+
4 CPU Cores
8 GB RAM
20 GB Free Disk Space
```

1. 准备 docker，docker-compose 环境
2. clone 项目

```
git clone https://github.com/getsentry/onpremise.git

cd onpremise
```

3. 初始化项目：`./install.sh`
